### 在校园
#### **`初入大学`** 


![20211216121703921.jpg](https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/54bb044090e14a0ab7dec5dabeaa351f~tplv-k3u1fbpfcp-watermark.image?)

&ensp;&ensp;&ensp;刚入学校的时候十分迷茫，甚至于不知道**计算机科学与技术**这个专业具体是干嘛的。
     
&ensp;&ensp;&ensp;这么多年也确信自己脑子属实愚笨，念书着实不行，不然也不会初中成绩差复读，高考就考了那么点分，在校期间甚至任何的校级荣誉和奖项都没拿过，唯一能说出手的事情，可能就是在图书馆做了两年的工读生，做到了文学区负责人（高级扫地人）领着一个月220块钱的学校补贴，带着同学们管理秩序和卫生，倒是闲着看了许多乱七八糟的文学区的书，后来兼职做家教，教一个初三的男孩子物理和化学，还有一个初一小姑娘的数学，男孩子自己很聪明，学的很快补习结束后考试也考得很好也是自己心仪的学校，到小姑娘时，效果不如自己预期，自己辅导经验很浅薄，又觉得没能力草率为人师觉得很惭愧，教了一段时间后就推辞自己要复习就不做了，钱也只象征性的收了一点。
     
&ensp;&ensp;&ensp;时间线很快来到了大三，遇到了一个很喜欢的专业课老师，跟着学了一段时间的JS和CSS，瞬间就被前端吸引了，看着网页上丑陋的布局，简易的效果，虽然当时身边的人都说后端更好找工作，但是当时真的觉得成就感满满，所以决心走前端的道路。
 #### **`一个大三学生`**   
 &ensp;&ensp;&ensp;郑州，一个让互联网新生从业者望而生退的城市，在郑州找开发方面的实习岗很难，真的很难。
 
 &ensp;&ensp;&ensp;大三末期，深知学校无法提供任何招聘机会的我，开始在招聘网站上大量的投简历，想提前进入实习阶段。
     
&ensp;&ensp;&ensp;简历倒是有回应，可是去过以后，多数要有实习经验才能实习，没有经验就交钱在那学习**培训机构**。有极少数心仪的公司也是要求有使用框架的经验，来了能干活。
     
   &ensp;&ensp;&ensp;实习工作的找寻使我屡屡受挫，开始怀疑自己，便在学校又开始了一轮新的学习，大三到大四开学的暑假期间，舍友都回了家，两个月的时间里，一个人在宿舍，租阿里云的服务器买域名搭网站，搭建了简易的个人网站，用来整理自己每天的学习笔记。

![image_1.png](https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/87d2742c08f54842aebc559aca34af7d~tplv-k3u1fbpfcp-watermark.image?)

![image_3.png](https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e5d9ca5e8c954a1293c59be0acacf2aa~tplv-k3u1fbpfcp-watermark.image?)
 #### **`一个实习生`** 
 
 &ensp;&ensp;&ensp;转折在蝉鸣将要结束的八月底，大四要开学了，当时某BOSS招聘APP上陆陆续续有一些联系，就去现场面了几个，当时亚信约了笔试和面试，结束的时候主管说，前端岗已经没坑位了，但是觉得我基础还挺好，就给了我一个后端的资格进来了（后边发现后端也没坑了，最后走的测试岗审批），就这样拿着每月2500的补贴就开始了实习。

![image.png](https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bbf873bca0324dadae942fd2523ab295~tplv-k3u1fbpfcp-watermark.image?)
&ensp;&ensp;&ensp;2019年9月，开启了我人生中的首次工作生涯，实习的初始阶段总让人感到新奇，不过再满腔的热情也败在了日复一日的收发邮件整理表格，同事说这样的生活起码要持续两个月，无法写代码属实难耐，于是我便自己开始整理大三时做的demo，用Vue再重构以期当做毕设。

&ensp;&ensp;&ensp;中间自己多次和领导进行沟通，以期能尽快调到客户开发现场写代码，终于成功进入中移，其中艰辛难以言表。进入开发现场以后，几十上百人乌泱泱的挤在光秃秃的桌子上，后来和同事沟通才知道，原来这是给客户当**外包**，并且这里不会有人是所谓的‘实习生’，其中缘由不再赘述。

&ensp;&ensp;&ensp;因为我需求的原因，便调离了公司的团队，直接和中移的人对接，身边不再有一个公司同事，直到后期我实习结束，其实在我整个实习生涯中，从来没有在亚信同事上学过任何技术上的东西，都是中移的同事们在耐心教我各种规范，各种技术和各种工具，也是他们让我养成了良好的代码规范，真的十分感激他们。

&ensp;&ensp;&ensp; **年关将至，疫情爆发，从道听途说，到眼前布满恐慌的人群，再到村村户户挖路封桥，肃穆的气氛扰了我的心境，乱了我的计划，年后公司无限期的延长了远程办公的时间，事实上直到我毕业，为了学生安全，实习生一直是远程。**

&ensp;&ensp;&ensp;2020年2月3日，我是第一批复工的人员，当初上报是否在家是否带笔记本时，我也曾犹豫，因为很多实习生为了避免远程都上报未带，但是有个东西我做了好久好久了，都快发版了移交给别人实属不甘...，我在元宵节也如愿经历了人生第一次发版，其中守夜的成就感不言而喻，没想到这次远程，我竟一下远程到毕业。

&ensp;&ensp;&ensp;5月已至正值实习期末，公司开始转聘答辩，我们也开始大批量的请假做校内部分收尾事宜，做的需求多加上答辩发挥好，实习中期和实习末期两次技术笔试加答辩总成绩排名侥幸都是头列，也拿到了转正名额，但是最后我拒绝了转聘，毅然决定和朋友前往杭州。

&ensp;&ensp;&ensp;拒绝郑州的转聘有几个原因，待遇不好是一方面，，另一方是大学都在家乡读的我，对外面世界的渴望，还有是深知郑州作为码农行业的局限性，心有不甘，现在出去能回来，不出去的话以后就难了。

![yaxin.jpg](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e5da15d3044b4e89b9bd29f3a8cb00bb~tplv-k3u1fbpfcp-watermark.image?)
&ensp;&ensp;&ensp;我提离职的那天，项目组一个很好的技术主管知道后和我聊天，鼓励我说趁年轻出去跑跑也好，长长见识学学东西，以后从外边想回来也容易。极少谋面的直接主管（业务）对我说：**我见过太多你们这种人，(他从毕业就在公司，待了七年)自认为学了点东西，出去后再也找不到这么好的公司了，自己想清楚，现在是有机会，等你们出去找不到工作再回来公司就不一定要你们了**。

&ensp;&ensp;&ensp;无论是暑假在校的两个月，还是因为疫情独自在郑州闭门那几个月远程的生活，包括后来毕业后我异地工作的大半年，身边没有朋友，常常独来独往，其实没有人喜欢孤独我也是，不过不同的是可能我慢慢变得比较擅长。
